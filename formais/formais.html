Description:

Automata® Minimum is a program that minimize a given automaton (drawed or by file). Examples of valid files can be download here. If the automaton is deterministic then the program try to minimize it, else a message is given describe the problem. Example, try to the file ndfa.aut in the program (It's a nondeterministic automaton).


How to use the AUTOMATA® MINIMUM:

If you want to draw, just press CTRL+N or click in File and New. Or you can open a file pressing CTRL+O or clicking in File and Open. To minimize it, press CTRL+M or click in Edit Automaton and Minimize. A new windown having the minimized automaton is showed. Click and move the states to rearrenge the automaton. To create a new state, click in the state symbol at toolbar and click in the draw area. To define a state as initial, click in the initial icon in toolbar and click in the state that you want to be the initial. Idem to create a final state.


How to Install the AUTOMATA® MINIMUM:

Download the executable (see Download) and descompress it into a directory of your own. To execute, make sure that you have the JRE 1.4.1 installed in your machine before run it. In the command line type "java -jar automato.jar", into the directory that you installed the program.
